# VAYU HOTAS Reference

CMDR ABHYUTHANAM's control mapping from `NEELAYODYSSEY.4.1.binds`.

## Devices

| Device | Type | Description |
|--------|------|-------------|
| ThrustMasterWarthogThrottle | Throttle | Thrustmaster HOTAS Warthog Throttle |
| VKB/GamePad | Joystick | VKB NXT EVO (Gladiator NXT) |

## Binding Categories

The bindings file contains mappings organized by these categories:

### Flight Controls
- **Throttle** — Throttle axis, forward/reverse
- **Yaw/Pitch/Roll** — Primary flight axes
- **Lateral/Vertical Thrust** — Translational movement
- **Flight Assist** — Toggle/hold FA off
- **Supercruise** — Engage/disengage supercruise
- **Hyperspace** — FSD jump
- **Landing Gear** — Deploy/retract

### Combat
- **Primary Fire** — Trigger group 1
- **Secondary Fire** — Trigger group 2
- **Next/Previous Fire Group** — Cycle weapon groups
- **Deploy Hardpoints** — Toggle weapons
- **Chaff** — Chaff launcher
- **Heat Sink** — Heat sink launcher
- **Shield Cell** — Shield cell bank
- **Boost** — Engine boost

### Targeting
- **Select Target Ahead** — Lock forward target
- **Next/Previous Target** — Cycle targets
- **Next/Previous Hostile** — Cycle hostile targets
- **Next/Previous Subsystem** — Cycle subsystems
- **Wingman Targets** — Lock wingman targets

### Power Distribution
- **Increase/Decrease SYS** — System power
- **Increase/Decrease ENG** — Engine power
- **Increase/Decrease WEP** — Weapon power
- **Balance Power** — Reset to 2/2/2

### UI Navigation
- **Galaxy Map** — Open galaxy map
- **System Map** — Open system map
- **UI Panel Left/Right/Up/Down** — Navigate panels
- **UI Select** — Confirm selection
- **UI Back** — Go back
- **Headlook** — Free look mode

### FSS (Full Spectrum System Scanner)
- **Enter FSS** — Open scanner
- **Zoom In/Out** — Frequency zoom
- **Tune Left/Right** — Frequency tuning
- **Target Current Signal** — Lock signal

### On-Foot (Odyssey)
- **Primary Interact** — Use/interact
- **Secondary Interact** — Alternative action
- **Sprint** — Run
- **Crouch** — Crouch/stealth
- **Jump** — Jump
- **Torch** — Toggle flashlight
- **Night Vision** — Toggle NV
- **Primary Fire** — Weapon fire
- **Aim Down Sights** — ADS toggle
- **Reload** — Reload weapon
- **Grenade** — Throw grenade
- **Shield Regen** — Energy shield

## Modifier Layers

Many buttons use modifier combos for expanded functionality. The Warthog throttle's switches and VKB hat positions serve as modifier layers.

## Unbound Functions

Some functions may be unbound by default. Check the Bindings page in VAYU at `/bindings` for a complete visual mapping with unbound functions highlighted.

## Viewing Bindings

The HOTAS Binding Visualizer at `/bindings` provides:
- SVG rendering of both devices
- Color-coded category highlighting
- Search/filter by function name
- Modifier layer visualization
- Unbound function detection
